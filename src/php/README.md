## Building PHP from source

**Default build**
- `wget https://get.webino.org/php -qO- | sh`

**Custom PHP version**
- `wget https://get.webino.org/php -qO- | sh -- -php 7.1.17`

**PHP-FPM support**
- `wget https://get.webino.org/php -qO- | sh -- -php-fpm`
